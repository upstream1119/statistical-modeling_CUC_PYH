## 数据收集要求

1. 每个数据源必须记录下载时间、来源链接或数据库名称。
2. 省份名称、年份范围、单位口径必须写清楚。
3. 如果数据经过换算、取对数、缩尾或插值，需要在处理说明中记录。
4. 不要直接上传无法确认授权的大型商业数据库文件。
