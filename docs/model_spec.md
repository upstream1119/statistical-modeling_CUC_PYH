# 冲奖版建模规格说明

本文题目为《数字普惠金融驱动区域低碳转型的空间溢出效应与政策优化研究》。当前建模目标已经从“跑通基准回归”升级为“冲击省一/国奖的完整统计建模方案”。因此，模型路线不能停留在单一显著性检验，而应围绕数据可信度、空间依赖、区域异质性、机制识别、政策分型和图表表达形成闭环。

## 1. 研究主线

当前结果不支持把数字普惠金融解释为全国层面的稳定直接降碳因素。更高标准的论文主线应为：

> 省域碳强度存在稳健空间依赖，数字普惠金融的全国平均直接效应方向不稳定，其低碳作用更可能依赖区域发展阶段、地理邻近关系和空间联系结构。本文进一步通过机制路径检验、政策分型和图表化呈现，将结果转化为分区协同治理建议。

这条主线允许我们诚实面对核心变量不完全显著的问题，同时把论文从“结果显著性论文”升级为“结构识别与政策优化论文”。

## 2. 正式样本与变量

正式样本为 2011-2022 年中国大陆 30 个省级地区，共 360 条省份-年份观测。不含西藏、港澳台；西藏剔除原因是 CEADs 表观碳排放清单未提供西藏地区数据，无法构造被解释变量。

核心字段如下：

| 类型 | 字段名 | 含义 |
| --- | --- | --- |
| 被解释变量 | `carbon_intensity` | 碳强度，万吨 CO2 / 亿元实际 GDP |
| 核心解释变量 | `digital_finance` | 北京大学数字普惠金融省级综合指数 |
| 控制变量 | `gdp_per_capita` | 人均实际 GDP |
| 控制变量 | `industrial_structure` | 第二产业增加值占现价 GDP 比重 |
| 控制变量 | `urbanization` | 城镇人口占总人口比重 |
| 控制变量 | `government_intervention` | 财政支出占现价 GDP 比重 |
| 控制变量 | `innovation` | `ln(1 + patent_grants)` |
| 控制变量 | `energy_structure` | 煤炭及煤炭加工相关终端能源消费折标准煤占比 |

## 4. 冲击省一/国奖的必补模块

现有模型足以完成一篇稳妥论文，但若目标是省一或国奖，需要继续补三个高价值模块。

### 4.1 机制路径检验

机制检验不写成严格因果中介，而写成“机制路径检验”。建议优先检验三条路径：

1. `digital_finance -> innovation -> carbon_intensity`
2. `digital_finance -> industrial_structure -> carbon_intensity`
3. `digital_finance -> energy_structure -> carbon_intensity`

判断重点不是每条路径都显著，而是说明数字普惠金融可能通过创新、产业结构和能源结构影响低碳转型，并解释为什么全国平均效应可能不稳定。

### 4.2 政策分型

政策分型是统计建模大赛中提升作品完整度的关键。建议使用省份均值或趋势变化构建四象限或聚类分型：

1. 高数字金融-低碳强度：示范引领型
2. 高数字金融-高碳强度：转型压力型
3. 低数字金融-低碳强度：稳态优化型
4. 低数字金融-高碳强度：重点扶持型

若时间允许，可进一步结合区域分组和空间权重矩阵结果，提出差异化政策建议。

### 4.3 图表升级

冲高奖必须让评审快速看懂结果。建议至少补充：

1. 碳强度年度趋势图
2. 数字普惠金融年度趋势图
3. Moran's I 年度变化图
4. 区域异质性系数图
5. 三类空间权重矩阵下 `W_digital_finance` 对比图
6. 政策分型散点图或象限图

## 5. 写作红线

以下内容在完成前不能写成既成事实：

| 不得写法 | 正确写法 |
| --- | --- |
| 数字普惠金融已经显著降低全国碳强度 | 全国平均效应方向不稳定 |
| 空间溢出效应已经稳健显著成立 | 地理距离矩阵提供边际负向证据 |
| 所有稳健性均支持核心结论 | 结果对模型设定和权重矩阵较敏感 |
| 机制检验已经证明完整因果链 | 后续将进一步检验机制路径 |
| 政策分型已经完成 | 政策分型是冲高奖下一步必做模块 |
