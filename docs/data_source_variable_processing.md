# 数据来源与变量处理说明

本文档用于回应数据可信度要求，可直接交给徐若诚作为论文变量说明和数据说明的基础材料。

## 1. 样本范围

正式建模样本为 **2011-2022 年中国大陆 30 个省级地区**，共 360 个省份-年份观测值。

正式样本不含西藏、香港、澳门、台湾。其中，西藏剔除的直接原因是 **CEADs 表观碳排放清单未提供西藏地区的省级表观碳排放数据**，导致被解释变量无法构造。含西藏的合并审计版保留在 `data_processed/panel_data_with_tibet_audit.csv`，仅用于说明缺失原因，不进入正式建模。

## 3. 关键变量口径说明

### 3.1 碳强度与 GDP

当前不再使用现价 GDP 直接计算碳强度。`gdp_real` 由国家统计局地区生产总值指数链式折算，基期为 2011 年。因此论文中应写：

```text
本文以省级碳强度衡量区域低碳转型水平，计算公式为
carbon_intensity = carbon_emissions_mt * 100 / gdp_real。
其中，carbon_emissions_mt 来自 CEADs 表观碳排放清单，单位为百万吨 CO2；
gdp_real 为以 2011 年为基期折算的实际地区生产总值，单位为亿元。
因此，碳强度可解释为万吨 CO2 / 亿元实际 GDP。
```

### 3.2 能源结构

此前“原煤相关碳排放占总表观碳排放比重”的口径已经替换。当前采用更稳妥的能源消费结构代理变量：

```text
energy_structure = coal_related_tce / total_energy_tce
```

其中，`coal_related_tce` 为原煤、洗精煤、其他洗煤、型煤、焦炭等煤炭相关终端能源消费折标准煤后的合计值，`total_energy_tce` 为所有终端能源消费折标准煤后的合计值。该指标应表述为“煤炭相关终端能源消费占比”或“煤炭依赖程度”，不要写成严格意义上的一次能源消费结构。

个别能源品种在《中国能源统计年鉴2022》附录4中给出区间系数或比 CEADs 分类更细，当前按区间中点或最接近类别处理，并已在 `code/01_data_cleaning.py` 中记录。

### 3.3 创新水平

创新水平不再直接使用专利授权量原值，而是使用：

```text
innovation = ln(1 + patent_grants)
```

这样可以缓解专利授权量的右偏分布和省份规模差异。`patent_grants` 原值仍保留在数据中，方便审计和稳健性检查。

## 4. 空间权重矩阵确认

当前空间权重矩阵文件为 `data_raw/spatial_weights.csv`，字段为：

```text
province_i, province_j, weight
```

矩阵口径如下：

1. **矩阵类型：** 0/1 省级邻接矩阵。
2. **构造方式：** 两个省级地区接壤则 `weight = 1`，否则不记录或视为 0。
3. **样本匹配：** 矩阵覆盖的省份与 `data_processed/panel_data.csv` 中的 30 个正式样本省份完全一致。
4. **孤立点处理：** 每个省至少有一个邻接关系。海南按与广东相邻处理，避免空间模型中出现孤立点。
5. **行标准化：** 原始 CSV 不做行标准化，后续空间计量代码读取后再进行行标准化。

已完成的审计结果：

```text

```

## 5. 已完成的稳妥化处理

针对三类风险，当前处理如下：

1. **能源结构：** 已从“原煤相关碳排放占比”改为“煤炭相关终端能源消费折标准煤后占总终端能源消费比重”。这更接近能源消费结构，但论文仍应谨慎表述为“煤炭依赖程度的代理变量”。
2. **GDP：** 已补入国家统计局地区生产总值指数，并折算为 2011 年基期实际 GDP。碳强度和人均 GDP 均使用实际 GDP 口径。
3. **创新水平：** 已采用 `ln(1 + 国内发明专利授权量)`，同时保留专利授权量原值作为审计字段。

## 6. 仍需如实说明的局限

1. 西藏不是人为随意删除，而是因为 CEADs 表观碳排放清单缺失西藏，正式样本无法构造被解释变量。
2. 能源结构虽然已改为折标准煤后的煤炭相关能源消费占比，但仍是能源结构的代理变量，不应写成完整、严格的一次能源结构。
3. 空间权重矩阵当前为邻接矩阵，后续可使用地理距离矩阵或经济距离矩阵做稳健性检验。
