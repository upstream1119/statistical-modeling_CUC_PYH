# 冲奖版文献调用与写作执行规范

本文件用于指导论文写作和文献引用。当前目标是冲击省一/国奖，因此文献综述不能只是罗列前人研究，而要服务于本文的核心发现：数字普惠金融的低碳作用并非全国同质化直接效应，而是具有空间依赖、区域异质性和空间结构依赖。

## 0. 总体导向

论文写作必须遵循以下主线：

> 省域碳强度存在显著空间依赖；数字普惠金融的全国平均直接效应方向不稳定；地理邻近提供边际负向外溢证据；区域异质性说明数字普惠金融低碳作用依赖地区发展阶段和产业基础。冲击更高奖项需要进一步补充机制路径检验、政策分型和高质量图表。

这意味着文献不能只服务于“数字普惠金融降低碳强度”这一单线命题，而要支撑四个层次：

1. 数字普惠金融与低碳转型的理论关联。
2. 碳排放和低碳转型的空间依赖。
3. 区域异质性与发展阶段差异。
4. 机制路径与政策分型。

## 1. 文献调用优先级

| 优先级 | 文献类型 | 用途 |
| --- | --- | --- |
| A | 数字普惠金融与碳排放、碳强度、绿色发展 | 支撑选题合理性和理论机制 |
| A | 空间计量、空间溢出、区域低碳转型 | 支撑 Moran's I、SAR、SDM 和空间权重矩阵 |
| B | 技术创新、产业结构、能源结构机制 | 支撑后续机制路径检验 |
| B | 区域异质性、东中西差异、东北产业锁定 | 支撑分区解释 |
| B | 综合评价、聚类分型、政策优化 | 支撑后续政策分型模块 |
| C | 绿色金融、数字经济、资源错配、绿色全要素生产率 | 用于丰富理论背景和政策建议 |

## 2. 各章节写作逻辑

### 2.1 摘要

摘要必须避免过度承诺。推荐结构：

1. 说明研究背景：数字普惠金融与区域低碳转型。
2. 说明方法：省级面板数据、Moran's I、空间计量、异质性和权重矩阵稳健性。
3. 说明主要发现：空间依赖稳健存在，全国平均效应不稳定，地理邻近提供边际负向证据。
4. 说明政策含义：区域协同、分区施策、地理邻近扩散。

不要在摘要中写成强单向结论。

### 2.2 引言

引言建议按三段展开：

1. 双碳目标下，区域低碳转型存在明显空间关联，单一地区政策难以孤立发挥作用。
2. 数字普惠金融可能通过缓解融资约束、促进创新、推动产业升级和优化能源结构影响低碳转型。
3. 现有研究多关注平均效应，本文强调空间依赖、区域异质性和空间权重结构敏感性。

### 2.3 文献综述

文献综述建议分为四类：

1. 数字普惠金融与绿色低碳发展。
2. 碳排放、碳强度与空间溢出。
3. 数字金融影响低碳转型的机制路径。
4. 区域异质性和政策分型研究。

最后要指出本文边际贡献：从省域尺度出发，将空间依赖、区域异质性、权重矩阵敏感性和政策优化连接起来。

### 2.4 理论机制与研究假设

当前建议提出谨慎假设，而不是强假设：

| 假设 | 内容 | 写作边界 |
| --- | --- | --- |
| H1 | 省域碳强度存在空间依赖 | 当前结果支持较强 |
| H2 | 数字普惠金融可能通过创新、产业结构和能源结构影响碳强度 | 需要后续机制检验支撑 |
| H3 | 数字普惠金融低碳作用存在区域异质性 | 当前东部边际负向提供线索 |
| H4 | 空间溢出效应对权重矩阵设定敏感 | 当前地理距离矩阵提供边际证据 |

## 3. 实证结果与文献的衔接

当前实证结果应这样调用文献：

| 当前结果 | 可调用文献方向 | 标准解释 |
| --- | --- | --- |
| Moran's I 稳定为正 | 空间集聚、区域碳排放联动 | 低碳转型具有空间关联，需要空间模型 |
| OLS 中核心变量不显著 | 区域异质性、非线性、门槛效应 | 全国平均模型可能掩盖差异 |
| SAR 中空间滞后显著 | 空间依赖和邻近效应 | 本地碳强度受周边地区影响 |
| SDM 中空间滞后项为负但不显著 | 空间外溢、地理扩散 | 只能写方向性线索 |
| 地理距离矩阵边际负向 | 地理邻近、知识扩散、政策扩散 | 地理邻近可能是重要传导渠道 |
| 东部边际负向 | 区域发展阶段、数字基础设施 | 东部更可能具备低碳赋能条件 |

## 4. 机制检验写作规范

后续若补机制检验，建议围绕三条路径：

1. 技术创新机制：数字普惠金融缓解融资约束，促进创新投入和技术扩散。
2. 产业结构机制：数字普惠金融改善资本配置，引导产业升级。
3. 能源结构机制：数字普惠金融支持绿色项目和能源替代，降低煤炭依赖。

如果机制检验结果不完全显著，也可以写成“部分机制路径得到方向性支持”，不要写成完整因果链已被证明。

## 5. 政策分型写作规范

后续若补政策分型，可使用四象限或聚类方法。若使用 TOPSIS 或 K-means，必须明确其只是政策分型工具，不是为了制造显著性。

建议分型：

1. 示范引领型：数字金融水平高、碳强度低。
2. 转型压力型：数字金融水平高、碳强度高。
3. 稳态优化型：数字金融水平低、碳强度低。
4. 重点扶持型：数字金融水平低、碳强度高。

政策建议必须对应分型，不要泛泛而谈。

## 6. 禁止表述

论文中不得出现以下意思的表述：

| 风险表述 | 替代表述 |
| --- | --- |
| 将全国平均模型解释成稳定降碳证据 | 全国平均效应方向不稳定 |
| 将空间滞后项解释成稳健外溢 | 地理邻近提供边际负向证据 |
| 将普通稳健性写成全部支持 | 稳健性结果提示模型设定敏感 |
| 将机制路径写成既成事实 | 后续将进一步进行机制路径检验 |
| 将政策分型写成已完成成果 | 政策分型是冲高奖下一步必做模块 |
| 将正式样本写成包含西藏 | 正式样本为中国大陆 30 个省级地区 |
| 将碳强度写成基于现价 GDP | 碳强度基于 2011 年基期实际 GDP |

## 7. 团队执行规则

徐若诚写作时优先使用《阶段性实证结果与论文叙事指南》和 `tables/table10_empirical_findings_summary.csv`。

严欣浩负责核对变量、数据来源和文献中涉及的数据口径。

彭意涵负责最终核对所有实证结论是否被论文文字写过头。

所有文献调用必须区分三类内容：理论推断、已有文献证据、本文实证发现。不能把文献中的城市层面或企业层面结论直接写成本文省级样本已经证明的结论。
