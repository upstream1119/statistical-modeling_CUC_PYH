# 空间权重矩阵与区域分组说明

本文件用于说明空间模型、权重矩阵稳健性和区域异质性分析的设定逻辑。冲击省一/国奖时，空间权重矩阵不能只是技术附件，而要服务于“空间结构依赖”的论文主线。

## 1. 空间权重矩阵体系

项目当前构建三类覆盖正式样本 30 个省级地区的空间权重矩阵。代码读取后统一进行行标准化。

| 矩阵 | 文件 | 论文功能 |
| --- | --- | --- |
| 邻接矩阵 | `data_raw/spatial_weights.csv` | 主模型矩阵，表示省份是否接壤 |
| 地理距离倒数矩阵 | `data_raw/spatial/spatial_weights_geo_distance.csv` | 检验地理邻近渠道 |
| 经济距离倒数矩阵 | `data_raw/spatial/spatial_weights_economic_distance.csv` | 检验经济相似关系 |

## 2. 邻接矩阵

邻接矩阵为 0/1 矩阵。若两个省级地区接壤，则 `weight = 1`，否则为 0。海南按与广东相邻处理，以避免空间模型中出现孤立点。

邻接矩阵是主模型矩阵，主要用于回答：本地碳强度是否受到相邻地区碳强度和相邻地区数字普惠金融水平影响。

当前主模型结果显示：

| 模型 | 关键项 | 结果 | 解释 |
| --- | --- | --- | --- |
| SAR | `W_carbon_intensity` | `0.3762, p < 0.001` | 碳强度空间依赖稳健存在 |
| SDM | `W_digital_finance` | `-0.0271, p = 0.1643` | 空间滞后项为负但不显著 |

## 3. 地理距离倒数矩阵

地理距离矩阵基于省级行政中心经纬度，使用 Haversine 公式计算球面距离，再取倒数构造权重。

该矩阵用于检验数字普惠金融低碳外溢是否更符合地理距离衰减逻辑。当前结果显示：

`W_digital_finance = -0.0903, p = 0.0945`

这说明地理距离矩阵下存在负向边际显著线索，可以作为“地理邻近渠道”的证据，但不能写成稳健显著空间溢出。

## 4. 经济距离倒数矩阵

经济距离矩阵基于 2011-2022 年平均人均实际 GDP 差异构造。经济差异越小，权重越大。

该矩阵用于检验数字普惠金融外溢是否沿经济相似关系传播。当前结果为：

`W_digital_finance = 0.0338, p = 0.2494`

结果不显著且方向转正，因此不能解释为支持低碳外溢效应。它的论文作用是说明空间溢出对权重矩阵设定敏感。

## 5. 四大地区分组

地区分组文件为 `data_raw/region_groups.csv`，按照国家统计局四大地区口径划分为东部、中部、西部、东北。正式样本中西部不含西藏，原因是 CEADs 缺失西藏碳排放数据。

当前区域异质性结果如下：

| 区域 | 结果 | 写作口径 |
| --- | --- | --- |
| 东部 | `coef = -0.0055, p = 0.0638` | 边际负向线索 |
| 中部 | 负向但不显著 | 不能写显著 |
| 西部 | 不显著 | 说明数字金融基础或产业条件可能不足 |
| 东北 | 显著为正但仅 3 省 | 只作谨慎现象讨论 |

## 6. 冲奖版解释主线

空间矩阵和区域分组共同支撑以下结论：

1. 碳强度存在稳定空间依赖。
2. 数字普惠金融的全国平均效应不稳定。
3. 地理邻近下存在边际负向外溢线索。
4. 区域差异解释了全国平均效应不稳定。
5. 政策建议应从全国统一政策转向分区协同政策。
